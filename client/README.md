# Community Discussion Forum with Real-Time Chat 🚀

A full-stack **real-time community discussion platform** built using the **MERN Stack** with **JWT Authentication**, **MongoDB Atlas**, and **Socket.IO** for live discussions and messaging.

This platform enables users to create discussions, engage in real-time conversations, comment on topics, manage profiles, and interact with a modern community-driven interface.

## 🚀 Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/yourusername/community-discussion-forum.git
```

### 2. Navigate to Project Folder

```bash
cd community-discussion-forum
```

### 3. Install Dependencies

#### Client

```bash
cd client
npm install
```

#### Server

```bash
cd server
npm install
```

---

## ⚙️ Environment Variables

Create a `.env` file inside the `server` folder:

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key
```

---

## ▶️ Run the Project

### Start Backend

```bash
cd server
npm run dev
```

### Start Frontend

```bash
cd client
npm run dev
```
